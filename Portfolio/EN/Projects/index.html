Encryption

This script first defines three functions: encrypt_file, decrypt_file and generate_key. Then it generates a random salt and a key from a password and a salt. Finally, it encrypts and decrypts a file.

Port Analyzer

This script defines a scan_port function that attempts to connect to a port on a given host. If the connection succeeds, the port is open. The script then uses this function to scan all ports from 1 to 1024 on the given host and prints the ports that are open.
