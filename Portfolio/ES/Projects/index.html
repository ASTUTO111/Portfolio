Tres en Raya

Hice este mini-juego en el módulo 319 en la escuela CsBe. El juego es un simple tres en raya; tiene un modo 1v1 en el que juegas con otra persona en el lugar y otro modo que es 1vIA en el que juegas con inteligencia artificial que tiene 3 modos: fácil, medio y difícil. Lo creé utilizando los lenguajes de programación HTML, CSS y JS. Aquí está mi enlace a GitHub si quieres ver más sobre el proyecto. Tres en Raya

Encriptación

Este script primero define tres funciones: encrypt_file, decrypt_file y generate_key. Luego genera una sal aleatoria y una clave a partir de una contraseña y una sal. Finalmente, encripta y desencripta un archivo.

Análisis de Puertos

Este script define una función scan_port que intenta conectarse a un puerto en un host dado. Si la conexión tiene éxito, el puerto está abierto. Luego, el script utiliza esta función para escanear todos los puertos del 1 al 1024 en el host dado e imprime los puertos que están abiertos.
