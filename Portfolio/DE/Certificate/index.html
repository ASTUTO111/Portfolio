Verschlüsselung

Dieses Skript definiert zunächst drei Funktionen: encrypt_file, decrypt_file und generate_key. Dann generiert es ein zufälliges Salz und einen Schlüssel aus einem Passwort und einem Salz. Schließlich verschlüsselt und entschlüsselt es eine Datei.

Port-Analyse

Dieses Skript definiert eine scan_port-Funktion, die versucht, eine Verbindung zu einem Port auf einem bestimmten Host herzustellen. Wenn die Verbindung erfolgreich ist, ist der Port geöffnet. Das Skript verwendet dann diese Funktion, um alle Ports von 1 bis 1024 auf dem angegebenen Host zu scannen und die geöffneten Ports auszugeben.
